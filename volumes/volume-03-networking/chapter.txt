📘 Volume 03 — Computer Networking
🎯 Objective

Build a deep understanding of computer networking from the physical connection between devices to secure communication over the Internet.

Outcome:
After completing this volume, you should understand how data travels between devices, how networks are designed, how common protocols work, and how to troubleshoot network issues.

Volume Challenge

Be able to explain the complete journey of a packet:

Application → TCP/UDP → IP → Ethernet → Switch → Router → ISP → Internet → Remote Server → Response Back

You should be able to identify what happens at each stage, which protocol is involved, and where common failures can occur.

Prerequisites for Volume 04

Before moving on, you should be able to:

Explain the OSI and TCP/IP models.
Understand IPv4 addressing and subnetting.
Configure basic network settings.
Explain how DNS, DHCP, HTTP, HTTPS, SSH, and TCP work.
Capture and analyze network traffic with Wireshark.
Troubleshoot common network issues.
Understand the role of firewalls, proxies, and load balancers.
Understand basic cloud networking concepts.
